# 💻 Stay Productive

This repository is dedicated to staying productive by combining native macOS features, **Shortcuts**, and **Apple Intelligence** to use our Macs exactly as Apple intended.

Instead of installing bulky, heavy third-party applications, we lean into macOS's native scripting, JXA (JavaScript for Automation), and on-device AI to automate daily tasks cleanly, lightweight, and efficiently.

---

## 🚀 Projects in this Repo

### 📂 [Beautiful Mac Folders](./beautiful-mac-folders)
Stop looking at boring blue folders. This workflow uses macOS Folder Actions and local Apple Intelligence to automatically categorize, color-tag, and assign expressive emojis/icons to your folders the moment they are created or renamed.

*   **Setup Script:** Automates Folder Action setup.
*   **Color System:** Semantic categorization (Finance = Green, Code = Purple, Media = Orange, etc.).
*   **Apple Intelligence Integration:** Auto-determines themes using local on-device models.

---

## 🛠 Philosophy
We believe the best productivity tools are the ones already installed on your Mac. By orchestration of Shortcuts, Shell, and Apple Intelligence, we can build custom workspaces that fit our development flow perfectly.
